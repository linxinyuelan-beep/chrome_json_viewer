Instructions

This page is for testing the Site Filter (Blacklist/Whitelist) functionality:

  1. Open the extension popup and go to the "Site Filter" tab
  2. Choose a filter mode:
    • Disabled: Extension works on all sites
    • Blacklist: Extension disabled on listed sites
    • Whitelist: Extension only works on listed sites
  3. Add the current site to the list
  4. Refresh the page and check if JSON detection works based on your settings
  5. Try hovering over the JSON text below or double-clicking it

Expected Behavior

  • When filter is disabled: JSON should be detected and highlighted on hover
  • When in blacklist mode and site is listed: No JSON detection should occur
  • When in blacklist mode and site is NOT listed: JSON detection should work normally
  • When in whitelist mode and site is listed: JSON detection should work normally
  • When in whitelist mode and site is NOT listed: No JSON detection should occur
