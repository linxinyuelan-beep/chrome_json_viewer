安装步骤

1

下载扩展

从 Chrome 网上应用店搜索 "JSON Formatter & Viewer" 或直接从项目仓库下载

2

开启开发者模式

在 Chrome 扩展管理页面 (chrome://extensions/) 开启右上角的"开发者模式"

3

加载扩展

点击"加载已解压的扩展程序"，选择项目的 public 目录

4

开始使用

访问包含 JSON 的网页，享受智能检测和格式化功能

使用方法

🎯 自动检测

扩展会自动检测并高亮显示网页中的 JSON 内容，双击即可在查看器中打开

🖱️ 悬停检测

使用 Ctrl+Shift+H (Mac: Cmd+Shift+H) 切换悬停检测模式

⌨️ 快捷格式化

选中任意文本后按 Ctrl+Shift+E (Mac: Cmd+Shift+E) 尝试格式化为 JSON

🔧 弹窗工具

点击扩展图标打开弹窗，可以直接输入 JSON 进行格式化、压缩等操作

📖 查看器功能

在 JSON 查看器中可以展开/折叠节点、复制内容、浏览历史记录

🕐 历史导航

查看器支持前进/后退导航，轻松浏览之前查看过的 JSON 数据
