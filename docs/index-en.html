Installation Steps

1

Install from Chrome Web Store (Recommended)

Visit the Chrome Web Store page and click the "Add to Chrome" button to install the extension

2

Install from Source Code (Developer Option)

Download the source code from the project repository, enable "Developer mode" on the Chrome extensions page (chrome://extensions/), click "Load unpacked", and select the project's public directory

3

Start Using

Visit web pages with JSON content and enjoy smart detection and formatting features
