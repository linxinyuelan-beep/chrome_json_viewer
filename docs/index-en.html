Installation Steps

1

Install from Chrome Web Store (Recommended)

Visit the Chrome Web Store page and click the "Add to Chrome" button to install the extension

2

Install from Source Code (Developer Option)

Download the source code from the project repository, enable "Developer mode" on the Chrome extensions page (chrome://extensions/), click "Load unpacked", and select the project's public directory

3

Start Using

Visit web pages with JSON content and enjoy smart detection and formatting features

How to Use

🎯 Auto Detection

The extension automatically detects and highlights JSON content on web pages. Double-click to open in viewer

🖱️ Hover Detection

Use Ctrl+Shift+H (Mac: Cmd+Shift+H) to toggle hover detection mode

⌨️ Quick Format

Select any text and press Ctrl+Shift+E (Mac: Cmd+Shift+E) to format as JSON

🔧 Popup Tools

Click the extension icon to open popup for direct JSON input, formatting, minification, and more

📖 Viewer Features

In the JSON viewer, expand/collapse nodes, copy content, and browse through history

🕐 History Navigation

The viewer supports back/forward navigation to easily browse previously viewed JSON data
