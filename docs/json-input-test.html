使用说明

这个 Chrome 扩展可以帮助您更方便地查看和格式化 JSON 数据。主要功能包括：

  • 自动检测页面上的 JSON 文本
  • 格式化并美化 JSON 数据
  • 提供树状结构视图
  • 支持大型 JSON 文件的高效处理
  • 保存 JSON 历史记录

您现在可以尝试使用扩展的弹出窗口输入功能来格式化自定义的 JSON 文本。只需点击浏览器工具栏中的扩展图标，切换到"JSON格式化"选项卡，然后粘贴您的 JSON 文本进行格式化。
